# 💡 Interactive Neon Shutter Portal

An advanced frontend authentication experience where UI meets physicality. Instead of a standard toggle, users must pull a virtual lamp cord to open a mechanical shutter and reveal the login portal.

## 🚀 Core Features
- **Pull-Lamp Toggle:** Interactive cord with pull-down physics.
- **Neon Border Animation:** 4-color cycling borders around the container and headings.
- **Mechanical Shutter:** CSS-driven sliding cover effect.
- **Glassmorphism:** Frosted glass UI with animated background shapes.
- **Functional Logic:** 
  - Register & Login (LocalStorage)
  - Password Show/Hide Toggle
  - Dynamic Password Reset with "Click to Copy" feature.

## 🛠️ Tech Stack
- **HTML5** (Semantic Structure)
- **CSS3** (Keyframe Animations & Flexbox)
- **Vanilla JavaScript** (DOM Manipulation)
